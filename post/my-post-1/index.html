本hugo博客的搭建和使用

Feb 22, 2023
1 minute read

1.下载hugo，我使用的是windows拓展版本

2.下载安装git，下载安装go，go需按照hugo说明下载依赖的对应版本

3.fork一个喜欢的hugo主题，本博客使用stack

4.将下载解压的hugo.exe文件放到想创建博客项目的文件夹如D:\xiaohuangblog

5.将D:\xiaohuangblog添加至系统环境变量path中，重启电脑

6.在D:\xiaohuangblog文件夹打开终端，键入命令创建博客项目

hugo new site myblog

7.删除myblog文件夹中的config.toml文件

8.键入命令进入根站点设置git项目初始化

cd myblog
git init

9.键入命令拉取主题模块

git submodule add http://github.com/HuangXinHao/hugo-theme-stack/ themes/hugo-theme-stack

10.将myblog\themes\hugo-theme-stack\exampleSite文件夹中的config.yaml文件复制到myblog文件夹下

PS：本地测试主题hugo server –theme=hugo-theme-stack –buildDrafts 浏览器里访问：http://localhost:1313/

11.创建一个github仓库命名HuangXinHao.github.io然后回到终端键入命令关联本地hugo和远程挂载仓库

hugo --theme=hugo-theme-stack --baseUrl="http://HuangXinHao.github.io/"
cd public
git init
git remote add origin https://github.com/HuangXinHao/HuangXinHao.github.io
git add -A
git commit -m "first commit"
git push -u origin master

在此之后更新文章，使用hugo生成新的静态页面，并使用git push进行同步

cd public
git add .
git status
git commit -m "add blog post"
git push
浏览器里访问：http://HuangXinHao.github.io/

本地创建新的草稿命令为hugo new post/newpost.md

本地生成新的静态页面hugo server –theme=hugo-theme-stack 或 hugo server –theme=hugo-theme-stack

本地发布草稿文章hugo命令带上 —buildDrafts或 -D
